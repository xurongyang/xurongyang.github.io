AQS注释

发表于 2017-03-02 |

AbstractQueuedSynchronizer.java

备注：

  1. node节点创建后waitStatus为0；
  2. 节点入队前如果队列是空的话，会创建一个空的头节点；
  3. 获取锁失败后会把waitStatus从0改为SIGNAL；
  4. acquire失败后会把当前节点的waitStatus从0修改为SIGNAL，无论是互斥还是共享；
